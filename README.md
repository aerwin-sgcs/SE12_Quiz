# SE12_Quiz
Repository for quiz questions for SE12

[Open this link to see the quiz](https://aerwin-sgcs.github.io/SE12_Quiz/NESA_SE_Quiz.html)

Working on making this web app 
  * teacher can add new questions at any point
  * students can keep track of the wrong and right answers
  * students can repeat any wrong answers until all questions are answered correctly
  * students compete with themselves
  * questions range in style - matching, drop and drag, T/F, multiple choice, short answers etc
